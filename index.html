Join the conference

One day conference on all things Groovy!

Warsaw Groovy User Group invites you to a one day, mini conference on all things Groovy!

GR8Day is a spinoff of GR8Conf, an independent, affordable series of conferences in Denmark, US and now in Poland. It’s dedicated to the technologies in the Groovy ecosystem.

And it’s free of charge!

Join us as a speaker or just come over to listen.

Technologies

Groovy

A powerful, optionally typed and dynamic language, with static-typing and static compilation, for the JVM, aimed at improving developer productivity thanks to a concise, familiar and easy to learn syntax

Grails

Web framework for the Java platform aimed at multiplying developersâ€™ productivity thanks to a Convention-over-Configuration, sensible defaults and opinionated APIs. Integrated ORM, Domain-Specific Languages, compile-time meta-programming and Asynchronous programming.

Gradle

Emerge from Build Hell today through Gradle, the modern open source polyglot build automation system. End build script chaos. End deathmarches. End bug regressions. End broken release processes.

Spock

The testing and specification framework for Java and Groovy applications. Beautiful and highly expressive specification language. Compatible with most IDEs, build tools, and continuous integration servers. Inspired from JUnit, RSpec, jMock, Mockito, Groovy, Scala, Vulcans, and other fascinating life forms.
